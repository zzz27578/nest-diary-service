# 小窝使用手册

## 登录

默认后台地址：

```text
http://服务器IP:28080
```

默认管理员密码：

```text
12345678
```

首次登录后请在“设置 -> 访问密钥”里修改管理员密码。

## 外部 API Key

外部 API Key 只给外部扩展使用，例如：

- AstrBot 插件通过 HTTP 访问独立小窝服务。
- MCP。
- 脚本。
- 第三方网页。

如果只使用未来的插件内置小窝模式，插件内部工具不需要 API Key。

启用方式：

1. 打开小窝设置页。
2. 生成或填写外部 API Key。
3. 勾选“启用外部 API 访问”。
4. 把这串 key 填入需要外部访问小窝的工具配置。

## 数据目录

当前模块化结构：

```text
data/
  system/settings/
  modules/diary/entries/
  modules/diary/index/
  modules/diary/snapshots/
  modules/impressions/people/
  modules/media/blobs/
  modules/media/by-date/
  user_custom/webui/themes/
  user_custom/webui/modules/
```

`user_custom/` 是用户或 bot 自己改页面、改模块的地方。官方更新不应覆盖这里。

从旧版升级时，小窝会把旧的 `diary/`、`memory/`、`media/`、`settings/` 中缺失的文件复制到新的模块化目录。旧目录不会被删除。

## 备份

设置页提供导入导出。

导出会优先包含：

```text
system/
modules/
user_custom/
imports/
```

导入兼容旧备份目录：

```text
diary/
memory/
media/
settings/
```

## 自定义前端

建议新建：

```text
user_custom/webui/themes/你的主题名/
```

只覆盖需要改的模板或 CSS，未覆盖的部分回退官方默认 WebUI。这样官方更新后，自定义页面不容易被覆盖。
