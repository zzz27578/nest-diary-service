# Changelog

## 0.3.0 - 2026-05-17

- Added diary edit and delete flow in the web panel.
- Added date-only archive jump selectors for existing diary years, months, and days.
- Added ZIP backup export and safe ZIP import for diary, memory, media, and settings data.
- Removed the visible revisions page from the web panel while keeping internal snapshots before overwrite/delete.
- Improved settings, access-token, and version management copy.
